Hotel Search Prototype

This is a hotel search website prototype using Expedia's API. It is a web app built with Sencha Touch framework.

Type in a location in the search box and it will give you a list of hotels in that area, including hotel name, a thumbnail image, address and ratings! Click into one of them will show the hotel on map.

It's a fun project I built in my free time.

WLPP Compiler

In a second year CS class, we designed and implemented a WLPP compiler in both C++ and Scheme. WLPP is a programming language that contains a subset of C++.
I implemented a scanner, a parser, a syntax , code generator based on the language specifications. The compiler was tested against large WLPP programs in school's testing environment, and the efficiency of the generated assembly programs were examined. Concepts of compiler and MIPS assembly language were applied.
